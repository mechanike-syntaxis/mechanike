Vis à tête cylindrique/fraisée cruciforme
=========================================

Présentation
------------

Les vis à tête cruciforme ont un faible couple de serrage, elles sont ulitisées pour des assemblages avec des faibles contraintes mécaniques. Il existe plusieurs types d'empreintes cruciformes, les plus utilisés sont le type Z aussi appelé "Pozidriv" et le type H aussi couramment appelé "Philippe".  

Exemple de designation:

- Une vis à tête bombée cruciforme: **ISO 7045 - M4 x 25 - 8-8 - Z**
- Une vis à tête fraisée cruciforme: **ISO 7046 - M4 x 25 - 8-8 - Z**
- Une vis à tête fraisée bombée cruciforme: **ISO 7047 - M4 x 25 - 8-8 - Z**

**M4** étant le diamètre de la vis, **25** la longueur, **8-8** la classe de qualité, **Z** ou **H** le type d'empreinte cruciforme.
